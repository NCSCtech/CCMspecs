Class FWHMCalibrationType

A data type for a FWHM calibration.

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,FWHMCalibrationType
Attribute cbrn:EnergyValueList

A list of energy values, in units of keV; the energies shall appear in the list in strictly increasing order. This element appears paired with an element that provides a corresponding list of other values, such as the EnergyDeviationValueList, FWHMValueList, or EfficiencyValueList elements. The number and order of corresponding values in the pair of lists must match.

Declaration :

  • Uml : - cbrn:EnergyValueList : EnergiesKeVType, multiplicity : 1,1

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,EnergyValueList
Attribute cbrn:FWHMValueList

A list of FWHM values, in units of keV. The number and order of corresponding values in the EnergyValueList and FWHMValueList lists must match.

Declaration :

  • Uml : - cbrn:FWHMValueList : FWHMKeVType, multiplicity : 1,1

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,FWHMValueList
Attribute cbrn:FWHMUncertaintyValueList

A list of the 1-sigma absolute uncertainties in units of keV, in the FWHM values contained in the FWHMValueList element list. The number and order of corresponding values in the FWHMValueList and FWHMUncertaintyValueList lists must match.

Declaration :

  • Uml : - cbrn:FWHMUncertaintyValueList : FWHMUncertaintiesKeVType, multiplicity : 0,1

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,FWHMUncertaintyValueList
Attribute cbrn:CalibrationDateTime

A date and time at which a calibration was put into service.

Declaration :
