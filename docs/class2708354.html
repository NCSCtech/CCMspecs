Class RadItemInformationType

A data type that provides information regarding an item from which radiation is being measured by the radiation instrument, and measurement data is reported in this xml document.

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,RadItemInformationType
Attribute cbrn:RadItemDescriptionText

A description of the item being measured.

Declaration :

  • Uml : - cbrn:RadItemDescriptionText : string, multiplicity : 0,1

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,RadItemDescriptionText
Attribute cbrn:RadItemQuantity

A count or size of the item being measured, and its uncertainty. The units and interpretation of this value will be application-specific, but will normally be the weight or volume of the measured item, used in the calculation of item activity concentration.

Declaration :

  • Uml : - cbrn:RadItemQuantity : RadItemQuantityType, multiplicity : 0,1

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,RadItemQuantity
Attribute cbrn:RadItemMeasurementGeometryDescriptionText

A description of the position and/or shape of the geometry used in the measurement of this item; e.g., shape of the item, item orientation relative to the radiation detectors, position of any attenuators used.

Declaration :

  • Uml : - cbrn:RadItemMeasurementGeometryDescriptionText : string, multiplicity : 0,1

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,RadItemMeasurementGeometryDescriptionText
Attribute cbrn:RadItemCharacteristics

A set of data providing characteristics of a measured item that are not otherwise explicitly defined in the relevant IEPD schema. Each characteristic consists of name, value, units, and value data class. Characteristics may also be organized in characteristic groups.

Declaration :

  • Uml : - cbrn:RadItemCharacteristics : CharacteristicsType, multiplicity : 0,unbounded

Properties:

  • URI:
    http://release.niem.gov/niem/domains/cbrn/5.0/,RadItemCharacteristics
