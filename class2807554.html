Class SectorType

A data type for a sector (i.e., major grouping of critical infrastructure) to which the asset belongs. There are 18 such Sectors as outlined in the National Infrastructure Protection Plan.

Properties:

  • URI:
    http://release.niem.gov/niem/domains/infrastructureProtection/5.0/,SectorType
Attribute ip:SectorName

A major grouping name of critical infrastructure to which the facility belongs.

Declaration :

  • Uml : - ip:SectorName : TextType, multiplicity : 0,unbounded

Properties:

  • URI:
    http://release.niem.gov/niem/domains/infrastructureProtection/5.0/,SectorName
Attribute ip:SubSector

A major grouping within the sector to which the asset belongs -see Infrastructure Taxonomy Version 3.

Declaration :

  • Uml : - ip:SubSector : SubsectorType, multiplicity : 0,unbounded

Properties:

  • URI:
    http://release.niem.gov/niem/domains/infrastructureProtection/5.0/,SubSector
Attribute ip:SectorTaxonomyID

An identifier of each sector as used in the Taxonomy -see Infrastructure Taxonomy Version 3.

Declaration :

  • Uml : - ip:SectorTaxonomyID : string, multiplicity : 0,unbounded

Properties:

  • URI:
    http://release.niem.gov/niem/domains/infrastructureProtection/5.0/,SectorTaxonomyID
