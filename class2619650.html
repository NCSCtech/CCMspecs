Attribute ag:GeospatialSourceDataCreationDateTime

A date when the data that was used as a source for geospatial data being considered. For geospatial data sourced by means of Precision Agriculture, this is the date the raw data was collected by the equipment. For geospatial data that was created from an existing Common Land Unit (CLU) shape, this is the date the source CLU was created.

  • Uml : - ag:GeospatialSourceDataCreationDateTime : dateTime, multiplicity : 1,1
